### 🔥 개발 목적

EMS의 항공편 정보 조회를 한 곳에서 해결할 수 없는 점, 
EMS의 현재 상황을 조회할 때마다 새로 등기번호를 입력해야 되는 점 등  
기존 우체국 어플리케이션의 EMS 조회 시스템의 **다양한** 불편점을 개선하여      
사용자들에게 보다 편리한 EMS 서비스를 제공하고자 개발하였습니다.
